6.1. Componentes

Relacionar Dispositivos con sus Categorías y Funciones

Objetivos:
- Identificar los componentes básicos de un sistema IoT: dispositivos, gateways, nube, análisis de datos y aplicaciones.
- Comprender cómo estos elementos trabajan en conjunto para formar un sistema conectado y funcional.
- Reflexionar sobre la importancia del análisis de datos para transformar la información en acciones útiles.

Consigna:
Participa en este juego interactivo donde deberás relacionar los componentes clave de un sistema IoT con sus funciones y categorías.

- Asocia cada dispositivo con su rol en la estructura del IoT.
- Reflexiona sobre cómo los sensores envían datos a la nube a través de un gateway y cómo esta información se analiza para generar acciones.
- Comparte tus respuestas y reflexiones en clase cuando el docente lo indique.
